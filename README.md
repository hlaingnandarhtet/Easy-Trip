# Easy-Trip
Easy-Trip System is a modern, real-time Online Travel Booking Platform that allows users to instantly search, select, and reserve inter-city express bus tickets and hotel accommodations in a single interface.

Built using a decoupled Clean Architecture with .NET 8 (Web API) on the backend and a responsive Blazor WebAssembly (WASM) front-end powered by MudBlazor UI, the system delivers smooth single-page application performance. It leverages cloud-hosted Supabase (PostgreSQL) via Entity Framework Core Database-First approach to handle high-concurrency transactions, manage precise seat allocation states, and automate administrative inventory workflows effortlessly.
